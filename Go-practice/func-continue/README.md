### Multiple Return Values and Named Return Values

**Go functions are not restricted to returning a single value**


**Named Return Values**

**When to use Named Return Values**

**Named return values are particularly useful for:**

*    Documentation: They provide context about what each return value represents in the function signature.
    Complex Functions: In functions with multiple exit points (multiple return statements), they allow you to set the return values throughout the function and return them cleanly at the end.*
